# Cecex GitHub Pages Site

This is a simple static site designed for the `Cecex-git.github.io` repository. It is built with plain HTML, CSS, and JavaScript so it can be published directly with GitHub Pages.

## Files

- `index.html`: page structure and content sections
- `styles.css`: layout, colors, and responsive styling
- `script.js`: technologies and project data rendered on the page

## How to update the site

1. Open `script.js`.
2. Add or edit items in the `technologies` array.
3. Add more objects to the `projects` array for each repository you want to showcase.
4. Update the intro text in `index.html` if you want a different personal pitch.

## How to publish on GitHub Pages

1. Push these files to the root of `Cecex-git.github.io`.
2. In the repository settings, open **Pages**.
3. Set the source to the main branch and the root folder if it is not already configured.
4. Your site will be available at `https://cecex-git.github.io/`.
